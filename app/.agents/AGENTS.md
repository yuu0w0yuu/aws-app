# このプロジェクトは何か
AWS上でWebアプリケーションを構築することを通じた学習用リポジトリです

# 構成
`terraform`: インフラ用のTerraformコード
`app/test-app`: ECS Taskへの疎通性をテストするための簡易的なGoアプリケーションコード。基本的に通常の開発では利用しないため読み込みは不要
`app/demo-app`: Next.jsで構築する検証用アプリケーションのコード

# 全般的な指示

# 作業時の指示
- コードの修正や環境の変更を行う際は、必ず事前に修正計画をplanとして提示し、ユーザの承認を待ってから実行に移ること
- 実施の過程でエラーや想定外の事象に遭遇した場合、必ず何が起こったか、その次に何をしようとしているのかの情報を提示すること
